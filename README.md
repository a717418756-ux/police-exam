# 短線雷達 StockRadar Pro

風險優先的分層決策股票分析 PWA。台股（TWSE/TPEX）+ 美股（Yahoo），透過 Google Apps Script 後端代理，無 CORS 問題、免費、免 API Key。

---

## 一、部署步驟

### 1. 後端（Google Apps Script）
1. 開 https://script.google.com → 新增專案
2. 把 `Code.gs` 整份貼入
3. 部署 → 新增部署作業 → 類型「網頁應用程式」
4. **執行身分：我自己｜存取權：所有人**（這項務必設對）
5. 複製「網頁應用程式 URL」

### 2. 前端（GitHub Pages）
把以下檔案**全部放在 repo 根目錄同一層**（不要開子資料夾）：

```
index.html  styles.css  config.js  db.js
market.js   quant.js    formula.js  journal.js  app.js
sw.js       manifest.json
```

`Code.gs` 不用上傳（它在 Google 端執行）。

### 3. 首次設定
打開網頁 → 右下 📒 → ⚙️設定與備份 → 貼上 GAS 網址 → 儲存 → 測試連線。

## 二、版本管理（破快取機制）

與考試用 PWA 相同：**只改 `config.js` 的 `APP_VERSION` 一個數字**（+1），`sw.js` 會自動以新版號建立快取並清除舊快取，不必手動刪。

```js
const APP_VERSION = 7;  // 改版時 +1 即可
```

## 四、自創量化公式數學原理

三個公式皆為**原創組合**（市面無），用真實數學工具建構。目標：短線方向判斷與**大跌前警示**。非保證準確，務必配合停損。

### STI 訊號張力指數（統計學）

$$STI = \frac{\sum_i w_i \cdot \tanh(z_i)}{\sum_i w_i} \times 100$$

其中 $z_i = \dfrac{x_i - \mu_i}{\sigma_i}$ 為 Z 分數（標準分數）。

- **原理**：RSI、乖離、量能、波幅等指標量綱不同，先用 Z 分數標準化到同尺度；`tanh` 把極端值壓縮到 ±1，避免單一指標暴衝；最後加權平均成 −100~+100 的「多空張力」。
- 正值＝多方張力，負值＝空方張力。

### MFD 動量流變導數（微積分）

$$MFD = \alpha \cdot \frac{dP}{dt} + \beta \cdot \frac{d^2P}{dt^2}$$

- **一階導數** $dP/dt$：價格速度（趨勢方向）
- **二階導數** $d^2P/dt^2$：加速度（趨勢加強或衰竭）
- **關鍵**：價漲（一階>0）但二階轉負 → 上漲動能衰竭，常為轉折前兆。本系統 $\beta=3$ 放大加速度權重，專抓這種「漲不動」的頂部訊號。

### ECO 熵能轉折指標（資訊論）

$$ECO = \left(1 - \frac{H}{H_{max}}\right) \times 100, \quad H = -\sum_i p_i \log_2 p_i$$

- $H$ 為夏農熵：把近期漲跌幅分 5 桶成機率分布。
- **原理**：盤整時分布均勻、熵高、ECO 低；趨勢明確時分布偏斜、熵低、ECO 高。ECO 由低轉高＝混沌收斂、趨勢即將成形（突破前兆）。

### 崩跌預警（多因子共振）

為降低假警報，須**多個空方因子同時成立**才提高分數：

| 因子 | 加分 |
|------|------|
| MFD 加速度轉負（動能衰竭） | +30 |
| ECO 偏空成形 | +25 |
| 量價背離（價漲量縮） | +20 |
| STI 張力轉空 | +15 |
| 連漲6根過熱 | +10 |

分數 ≥60 高風險、≥35 中風險。**特別針對大跌前的動能衰竭與量價背離設計。**

### FUSION 融合總分

$$FUSION = \big(0.3\,\tilde{S} + 0.4\,\tilde{M} + 0.3\,\tilde{E} - 0.5\,penalty\big) \times 100$$

三公式各自 `tanh` 正規化後加權（MFD 權重最高，因對轉折最敏感），再扣除崩跌風險懲罰。崩跌高風險時直接以避險為優先。

## 五、專屬量化分數（樣本內回測加權）

對每檔股票回看歷史：某指標發出訊號後 5 天內，是否真的大漲/大跌（±3%）。命中率即權重，漲跌分開計算。

$$\text{大漲分數} = \frac{\sum (\text{買進訊號} \times \text{該指標看漲命中率})}{\text{訊號數}} \times 100$$

**重要**：樣本內回測有過擬合風險，過去準不代表未來準。樣本不足者標「⚠️信心低」。系統隨新資料滾動更新，非固定公式。
